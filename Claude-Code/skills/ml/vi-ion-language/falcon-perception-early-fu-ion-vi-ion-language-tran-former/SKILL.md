# Falcon Perception Early-Fusion Vision-Language Transformer

## When to Use

- Building or understanding scalable, unified vision-language models for open-vocabulary
  grounding and segmentation.
- Designing an autoregressive Transformer backbone that early-fuses image patches and
  text tokens.
- Implementing variable-length, coarse-to-fine instance prediction (bounding box →
  size → segmentation) with minimal output heads.
- Deploying efficient, single-backbone OCR systems for complex document layout and
  semantics.
- Addressing tasks needing high compositional, spatial, relational, and OCR-based
  scene understanding.

## Architecture Overview

- **Single Autoregressive Transformer Backbone**: Processes unified sequence of image
  patches and text tokens, sharing parameters from first layer.
- **Hybrid Attention Mask**:
  - Image tokens attend bidirectionally to all image tokens (like a vision encoder).
  - Text and task tokens attend causally to prior tokens plus the full visual prefix.
- **Chain-of-Perception Output Interface**:
  1. `<coord>` token predicts object center coordinates.
  2. `<size>` token predicts object size.
  3. `<seg>` token produces segmentation mask by dot product with upsampled image features.
- **Lightweight Specialized Heads**:
  - Coordinate & Size heads use Fourier feature encoding for precise spatial prediction.
  - Segmentation head generates high-res masks without Hungarian matching or separate
    mask-queries.

## Training Pipeline

- **Multi-Teacher Distillation Initialization**:
  - Distill from DINOv3 (ViT-H) for local features.
  - Distill from SigLIP2 for language-aligned features.
- **Large-Scale Data Preparation**:
  - 54M images, 195M positive expressions, 488M hard negatives.
  - Hierarchical clustering for uniform concept coverage.
  - Dense object descriptions generated by VLM.
  - Multi-model ensemble consensus and human verification for label quality.
  - 1:1 positive-negative ratio for presence calibration.
- **Three-Stage Training Recipe**:
  1. **In-Context Listing** (450 GT): Autonomous listing of scene inventory with full causal
     attention.
  2. **Task Alignment** (225 GT): Queries attend only to visual prefix, masking other
     queries, focused on presence and localization.
  3. **Long-Context Finetuning** (10 GT): Enable long sequence decoding for dense scenes.
- **Ablations Confirmed**:
  - Muon optimizer for heads (+4.8 macro-F1).
  - Raster ordering of instances (+10 macro-F1).
  - Gram feature regularization (+1.5 macro-F1).
  - Global loss normalization for FSDP.

## Evaluation Metrics and Benchmarks

- **SA-Co Open-Vocabulary Segmentation**:
  - Falcon Perception (0.6B params) achieves 68.0 Macro-F1, beating SAM 3 baseline (62.3).
  - Presence calibration remains a gap (MCC 0.64 vs. 0.82).
- **PBench Diagnostic Benchmark**:
  - Levels of prompt complexity: L0 (simple objects) to L4 (relations), plus Dense scenes.
  - Falcon Perception shows increasing performance gaps vs. SAM 3 at higher levels.
  - Excels at OCR-guided, spatial, relational, and crowded scene scenarios.

## Falcon OCR Extension

- A 0.3B-parameter early-fusion model trained from scratch, optimized for document understanding.
- Supports multi-column layouts, mathematical formulae, tables, handwriting, and scene text.
- Uses same hybrid attention and unified backbone; switches tasks via prompts.
- Training on diverse OCR datasets mixing synthetic and real-world documents.
- Competitive accuracy on olmOCR (80.3%) and OmniDocBench (88.6%), with highest throughput
  among open-source OCR models.

## Inference Features

- PyTorch FlexAttention backend supporting custom hybrid attention masks.
- Paged KV cache with virtual page tables for efficient memory usage.
- Continuous batching supports dynamic, multi-instance generation.
- CUDA graph capture and background tokenization for low latency.
- HR feature caching to avoid repeated upsampling for multiple queries on one image.
- Typical latencies: ~100ms prefill, ~200ms upsampling, ~50ms decode on H100.

## Practical Guidelines

- Use early-fusion design to avoid complex pipelines and improve scalability.
- Employ hybrid attention masking to balance bidirectional visual context with causal
  language decoding.
- Implement Chain-of-Perception coarse-to-fine sequence predictions to handle variable
  numbers of instances efficiently.
- Initialize vision encoder weights via multi-teacher distillation for stable training.
- Curate training datasets with balanced positive-negatives and multi-model consensus
  filtering.
- Optimize specialized heads with Fourier features and Muon optimizer for fine localization.
- Structure training into curriculum stages from scene listing to independent query answering
  to long context finetuning.
- Integrate inference efficiency techniques (paged cache, continuous batching) for practical
  deployment.
